# Obaro Labs — Business Calculator

Internal revenue planning tool for Obaro Labs LLC.

## Features

- **Client Builder** — randomize or manually set client service mixes, see monthly revenue and take-home in real time
- **Recurring Engine** — model care plan and marketing retainer income, see MRR, ARR, and business valuation
- **Tax Calculator** — accurate Georgia LLC self-employment tax breakdown with federal brackets, quarterly schedule, and deductions checklist
- **Annual Projection** — 12-month forecast with churn modeling, stacked bar chart showing one-time vs recurring revenue
- **Cost Margins** — full delivery cost breakdown per service with gross margin percentages
- **Scenarios** — conservative / realistic / optimistic growth path comparison side by side

## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `obaro-calculator`)
2. Upload these three files: `index.html`, `style.css`, `app.js`
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Your calculator will be live at `https://yourusername.github.io/obaro-calculator`

## Files

```
index.html   — Main application structure
style.css    — Full design system (dark + light mode)
app.js       — All calculation logic and chart rendering
```

## Tech Stack

- Vanilla HTML / CSS / JavaScript — zero build step, zero dependencies
- Chart.js 4.4.1 (loaded from CDN)
- Google Fonts: Syne, DM Mono, Instrument Sans

## Notes

- All figures are estimates for planning purposes only
- Tax calculations use 2024 federal brackets and Georgia 5.49% flat rate
- Consult a licensed CPA before making financial decisions
- This tool is for internal Obaro Labs use only
